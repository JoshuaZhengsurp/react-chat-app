# Chat-React-App

一个基于 Web 的可用的即时通信应用（起步阶段）。

## 技术栈

### 前端：
- React/Vite
- Zustand
- SCSS
- 国际化（i18n）
- Husky/Commitlint
- TypeScript
- Axios
- Socket.IO
- Nginx

### 后端：
- NestJS
- Socket.IO
- Redis
- Prisma
- Docker
